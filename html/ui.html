UI/UX Design Courses

UI / UX for Beginners

The fundamentals of user interface (UI) and user experience (UX) design are simplified into clear principles in this UI UX course. Let's start with the fundamentals: explain what UI and UX are and how important they are in designing products that are not only aesthetically pleasing but also simple and pleasurable to use. You will be guided through the fundamentals of UX design on this trip, which includes identifying user needs and developing designs that successfully address them.

Car Transport App in Figma

You will be able to prototype your user journeys in XD at the project's conclusion! We will finish the customer's basic e-commerce journey, which includes product selection and payment completion. This includes understanding how to share the prototype for your stakeholders to see, as well as animating engaging and sticky material.

Learn Figma UI/UX Design for Absolute Beginners (in Sinhala)

If you're new to Figma and want to master the fundamentals of UI/UX design, this course is ideal for you. All of the fundamental Figma functionalities are covered in this course, such as designing and modifying designs, interacting with layers and shapes, and working in real-time teamwork.

Introduction to UI and UX Design

The experience people have when engaging with digital products and applications is greatly influenced by the design of the user interface (UI) and user experience (UX). We will discuss the concepts and methods of UI and UX design in this course. Additionally, you will create your own interactive prototypes and wireframes. Gaining a foundational understanding of UI and UX can improve teamwork on projects and open up new job prospects.

UI/UX Design using Adobe XD

You will learn how to create design layers, lock and hide them, group them for clarity, and then ungroup them in this course. Learn how to align page elements, copy and paste appearances, and assign a team member to a design project using the "grid" tool. Next, we'll walk you through using Adobe XD shapes. This section teaches you how to draw different shapes, work with them using the "resize" tool, and add borders and fills.
